IntroductionÂ¶

kcsd2d is a set of Matlab scripts for reconstructing CSD distributions from LFP measurements taken on two-dimensional planes. The method carried out by the scripts is described in Potworowski et al. Neural Computation(2012) 24 541-575. One should bear in mind that the reconstructions are calculated under the assumptions described in section 2.1 of the paper.

kcsd2d provides a set of easy to use Matlab scripts for reading your data, calculating CSD reconstructions and viewing them in a convenient viewer. It does not include filtering of data. Reconstructions can be easily exported. For a brief intro to using the scripts see the Quick Start section.

If you want to go into more detail you should be aware that the scripts have been embedded in a Matlab class: The kCSD-class. Certain actions can be carried out by running kcsd2d class Methods and data can be exported by accessing the kcsd2d class Properties. To get help on Matlab classes click here.
